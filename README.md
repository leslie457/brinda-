API Courses – Node.js & Express
Ce projet est une API REST développée avec Node.js et Express.
Il permet de gérer une liste de cours (courses) en utilisant des données temporaires stockées en mémoire, sans base de données.
L’objectif est de mettre en pratique la création d’une API structurée, la gestion des routes, la validation des données et le respect des standards HTTP.


        fonctionalites
Recurer tout mes cours
Recuperer tout les cours par ID
Ajouter un nouveau cours 
 Modifier un cours existant 
Supprimer un cours 
Validation minimale des entrées 
Gestion des erreurs propre et cohérente
